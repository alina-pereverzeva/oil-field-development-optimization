# Oil Field Development Optimization

## Описание проекта

Проект посвящен оптимизации стратегии разработки нефтяного месторождения на основе вероятностного моделирования и оценки экономической эффективности.
Основная цель — подобрать оптимальное количество скважин, мощность производственного объекта и целевой уровень plateau rate так, чтобы максимизировать устойчивый показатель **P10 NPV**.

В модели учитываются неопределенности по запасам, коэффициенту извлечения, дебиту скважин, стоимости бурения и ставке дисконтирования. Для перебора и оптимизации параметров используется библиотека **Optuna**, а расчеты ускорены с помощью **Numba**.

## Цель проекта

Разработать модель, которая помогает выбрать наиболее эффективную стратегию разработки нефтяного месторождения с учетом неопределенности исходных параметров.  
Модель оценивает разные варианты по финансовому показателю P10 NPV и позволяет определить конфигурацию с оптимальным соотношением доходности и риска.

## Методология

* Вероятностное моделирование методом Монте-Карло
* Расчет чистой приведенной стоимости проекта
* Оценка риска с использованием показателя P10 NPV
* Оптимизация параметров разработки с помощью Optuna
* Ускорение численных расчетов с использованием Numba
* Визуальный анализ результатов

## Оптимизируемые параметры

* Количество скважин для бурения
* Размер производственного объекта
* Plateau rate как процент от запасов

## Основные результаты

Оптимальная стратегия:

| Параметр           |     Значение |
| ------------------ | -----------: |
| Количество скважин |           13 |
| Мощность объекта   |          100 |
| Plateau rate       |        12.5% |
| P10 NPV            | 196.66 млн $ |

## Что было сделано

1. Сгенерированы вероятностные сценарии для ключевых параметров месторождения.
2. Построена модель производственного профиля.
3. Рассчитан NPV для разных сценариев разработки.
4. Использован показатель P10 NPV для учета риска.
5. Проведена оптимизация параметров с помощью Optuna.
6. Построены графики для анализа чувствительности и сравнения стратегий.

Проект демонстрирует применение методов оптимизации, вероятностного моделирования и финансового анализа для принятия решений в нефтегазовой разработке.
Полученная модель позволяет сравнивать стратегии разработки месторождения и выбирать вариант с наилучшим соотношением доходности и риска.
